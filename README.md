# SerialScreen (SwiftUI)

Native macOS Serial Monitor app written with Swift + SwiftUI.

## Run
```bash
cd "/Users/elvin/Documents/My Projects/SerialScreen/SerialScreenSwift"
swift run SerialScreen
```

## Build and install to Applications
```bash
cd "/Users/elvin/Documents/My Projects/SerialScreen/SerialScreenSwift"
./scripts/build_macos_app.sh
```

App path:
- `/Applications/SerialScreen.app`
